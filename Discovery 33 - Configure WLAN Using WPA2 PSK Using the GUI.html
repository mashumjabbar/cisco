  • Procedure1:
    You will start at the Monitor screen of the WLC. Click WLANs from the top menu bar.
    You will see the WLANs screen appear. Notice you have WLAN 10 configured. You want to add a new WLAN to the WLC.
  • Procedure2:
    Notice the drop-down menu in the upper right side of the screen. It is already set to Create New. Click Go to create a new WLAN.
    The WLANs > New screen will appear.
  • Procedure3:
    On the WLANS > New screen, there are four options that you can modify. The first option is Type. The default setting is WLAN. This is what you want, so donâ€™t change this. The next option is Profile Name. In the Profile Name dialog box, type profile name 2xauth2 and press Enter.
    Note
    This is a recording of the activity performed on real equipment. In a real environment, you would not need to press Enter and submit every entry. You would first fill in all the information on the screen and only submit it in the end, using the Apply button.
  • Procedure4:
    Next, look at the SSID option. In the SSID dialog box, type 2xauth2 and press Enter.
  • Procedure5:
    Next, look at the ID option. This is drop-down menu and it is the WLAN ID. Click the down arrow and choose 20.
  • Procedure6:
    Verify that all the parameters on your screen are configured correctly. When you are satisfied with the configuration, click Apply.
  • Procedure7:
    Verify that the SSID was created. Notice the Status is unchecked. Check the Enabled check box to enable this WLAN.
  • Procedure8:
    Verify that all the parameters are set correctly. Go to upper right side of the screen and click Apply.
    Look at the screen. Some of the fields have been configured from the last step, others are default values.
    • The Profile Name and SSID should read 2xauth2.
    • The type should be WLAN.
    • The Status is Enabled and checked.
    • The Radio Policy should be set to All.
    • The Interface/Interface Group should be set to Management.
    • The Multicast Vlan Feature should be unchecked.
    • Broadcast SSID should be checked.
    • The NAS-ID should be none.
    When you have verified all the parameters, click Apply.
  • Procedure9:
    A pop-up window will appear telling you that changing the WLAN parameters will cause the WLAN to be momentarily disabled. Click OK.
  • Procedure10:
    Next, click the Security tab.
    You will see the following screen. Notice the default selection is for Layer 2 in the Security tabs. Also, notice the default Layer 2 Security setting is WPA + WPA2.
    Look under WPA + WPA2 Parameters. Ensure that the WPA2 Encryption-AES check box is checked. This is the default setting.
  • Procedure11:
    Click the Next button to get to the bottom of the Layer 2 Security screen.
  • Procedure12:
    Look at the Authentication Key Management area. Click the check box next to PSK.
  • Procedure13:
    When you check the PSK option you will see a dialog box open to the right of PSK Format. The default for the PSK Format is ASCII. Donâ€™t change that. In the Dialog box enter the PSK (the password): type Iforgot2 and press Enter.
    Note
    When you type the credential into the dialog box you will not see what you are typing, only dots. Type carefully.
  • Procedure14:
    When you are satisfied with your configuration, click Apply.
  • Procedure15:
    You will see another dialog box pop-up. Click OK.
  • Procedure16:
    You will be taken back to the General screen for this WLAN.
  • Procedure17:
    You have completed this activity when you have attained these results:
    • You have configured a new WLAN, WLAN ID and SSID.
    • You have enabled the SSID.
    • You have verified the Layer 2 Security defaults.
    • You have verified the PSK format and configured the PSK (password).
